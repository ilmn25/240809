# 2.5D Game Project

Codebase for my game project  
https://docs.google.com/document/d/1GyaPKQXzhyW1kznnLDgLdSXv6SFFDBAZ09G7YVcqS-0/edit?usp=sharing

## Main features
- 3D pathfinding algorithm based on A*
- Cubic map chunking system
- States and module system for all entities
- Fully functional save / load system
- Real time map generation and loading with multithread job system
